# Javi Calorías — prototipo local

Abre `index.html` directamente en el navegador. El prototipo funciona sin servidor y guarda los cambios en `localStorage`.

Incluye resumen diario, registro editable, biblioteca de alimentos, historial semanal, gráficas sencillas y objetivos configurables. Los datos iniciales están basados en las cuatro semanas del Excel.

Siguiente fase: catálogo real de productos, recetas con ingredientes, importación completa del Excel y sincronización en la nube cuando Sites esté disponible.
